Integrales Fundamentales
Potencia
∫ xⁿ dx = xⁿ⁺¹/(n+1) + C
Válido para n ≠ −1. El exponente aumenta en 1 y se divide entre el nuevo exponente.
Logarítmica
∫ 1/x dx = ln|x| + C
Caso especial de n = −1. El resultado es el logaritmo natural del valor absoluto de x.
Exponencial
∫ eˣ dx = eˣ + C
La función exponencial e^x es su propia antiderivada. Propiedad única.
Exponencial general
∫ aˣ dx = aˣ/ln(a) + C
Para cualquier base positiva a ≠ 1. Se divide entre el logaritmo natural de la base.
Seno
∫ sen(x) dx = −cos(x) + C
La integral del seno es el coseno negativo. Recordar el signo negativo.
Coseno
∫ cos(x) dx = sen(x) + C
La integral del coseno es el seno. Sin cambio de signo a diferencia del seno.
Tangente
∫ tan(x) dx = ln|sec(x)| + C
Resultado logarítmico. También puede escribirse como −ln|cos(x)| + C.
Constante
∫ k dx = kx + C
La integral de una constante es la constante multiplicada por la variable.
Derivadas Fundamentales
Potencia
d/dx [xⁿ] = n·xⁿ⁻¹
Baja el exponente como coeficiente y resta 1 al exponente. Regla más usada.
Constante
d/dx [c] = 0
La derivada de cualquier constante es cero. Sin dependencia de x.
Exponencial
d/dx [eˣ] = eˣ
La función eˣ es la única función que es igual a su propia derivada.
Logaritmo
d/dx [ln(x)] = 1/x
El logaritmo natural se convierte en el recíproco de x al derivar.
Seno
d/dx [sen(x)] = cos(x)
La derivada del seno es el coseno. Primer par del ciclo trigonométrico.
Coseno
d/dx [cos(x)] = −sen(x)
La derivada del coseno es el seno negativo. Notar el cambio de signo.
Regla del producto
(f·g)' = f'g + fg'
Para derivar un producto: derivada del primero por el segundo, más el primero por la derivada del segundo.
Regla de la cadena
d/dx [f(g(x))] = f'(g(x))·g'(x)
Derivar la función exterior evaluada en la interior, multiplicada por la derivada de la interior.
Métodos de Integración
Por Sustitución
∫ f(g(x))·g'(x) dx = ∫ f(u) du
Se identifica u = g(x), se calcula du = g'(x)dx y se reemplaza para simplificar la integral.
Por Partes
∫ u·dv = uv − ∫ v·du
Útil para productos de funciones. Usar regla LIATE para elegir u: Logarítmica, Inversa trig, Algebraica, Trig, Exponencial.

Derivada: Regla de la Potencia Derivada

La regla más fundamental del cálculo diferencial. Permite derivar cualquier monomio de la forma axⁿ.

  • Identifica el coeficiente a y el exponente n en la expresión axⁿ.
  • Multiplica el coeficiente por el exponente: nuevo coef = a × n
  • Resta 1 al exponente: nuevo exp = n − 1
  • Escribe el resultado: (a·n)·x^(n−1)
  • Si n = 0 (constante), la derivada es 0. Si n = 1, la derivada es simplemente a.
Ejemplo: d/dx [5x³]
① Coeficiente = 5, Exponente = 3
② Nuevo coef = 5 × 3 = 15
③ Nuevo exp = 3 − 1 = 2
④ Resultado: 15x²
Integral: Regla de la Potencia Integral Simple

Operación inversa a la derivada. Aplica cuando la función es un monomio axⁿ con n ≠ −1.

  • Identifica el coeficiente a y el exponente n en axⁿ.
  • Verifica que n ≠ −1. Si n = −1, la integral es a·ln|x| + C.
  • Suma 1 al exponente: nuevo exp = n + 1
  • Divide el coeficiente entre el nuevo exponente: nuevo coef = a / (n+1)
  • Escribe el resultado y agrega la constante de integración +C.
Ejemplo: ∫ 4x² dx
① Coeficiente = 4, Exponente = 2
② n ≠ −1 ✓
③ Nuevo exp = 2 + 1 = 3
④ Nuevo coef = 4 / 3 = 4/3
⑤ Resultado: (4/3)x³ + C
Integral por Sustitución (u-sub) Sustitución

Técnica que simplifica integrales reconociendo una función compuesta. Es la versión integral de la regla de la cadena.

  • Identifica la función "interior" g(x) que complica la integral. Llámala u = g(x).
  • Calcula la diferencial: du = g'(x) dx, despeja dx.
  • Sustituye g(x) por u y dx por la expresión en du en la integral original.
  • Resuelve la integral simplificada en términos de u.
  • Sustituye de regreso: reemplaza u por g(x) en el resultado.
  • Añade la constante de integración +C.
Ejemplo: ∫ 6x(3x² + 1)² dx
① u = 3x² + 1
② du = 6x dx → dx = du/(6x)
③ Integral → ∫ u² du
④ = u³/3 + C
⑤ Resultado: (3x² + 1)³ / 3 + C
Integral por Partes Por Partes

Técnica para integrar productos de funciones. Se basa en la regla del producto para derivadas. Fórmula: ∫ u·dv = uv − ∫ v·du

  • Usa la regla LIATE para elegir u: Logarítmica → Inversa trig → Algebraica → Trigonométrica → Exponencial. La primera que aparezca en la lista será u.
  • Asigna el resto de la expresión como dv.
  • Calcula du derivando u, y calcula v integrando dv.
  • Aplica la fórmula: ∫ u·dv = u·v − ∫ v·du
  • Evalúa la nueva integral ∫ v·du. Si aún es complicada, aplica por partes de nuevo.
  • Simplifica y añade +C.
Ejemplo: ∫ x·eˣ dx
① LIATE: x es Algebraica, eˣ es Exponencial → u = x
② dv = eˣ dx
③ du = dx  |  v = eˣ
④ = x·eˣ − ∫ eˣ dx
⑤ Resultado: x·eˣ − eˣ + C = eˣ(x−1) + C
